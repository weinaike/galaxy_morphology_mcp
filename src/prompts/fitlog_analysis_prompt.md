## 分析任务

你将收到一张残差图像，包含三组图块（Original | Model | Residual），请先仔细观察图像，然后结合上述拟合日志进行综合诊断。

### 第一步：残差图像分析

请先对残差图像进行逐一观察：
- 残差图像是否存在系统性结构（如环形残差、条状残差、中心正/负残差等）？
- 模型图像与原始图像在形态上是否存在明显差异？
- 残差中是否存在未被拟合的额外光源？

### 第二步：参数异常检查

请按照以下三个类别，结合残差图像的观察结果，逐一检查上述拟合日志。

### 1. 参数简并检查
- 检查是否存在中心位置接近的成分对
- 比较它们的星等、有效半径是否存在"此消彼长"的简并关系
- 报告简并的严重程度

### 2. 不确定度异常检查
- 对每个成分的每个结构参数，检查相对不确定度 sigma/value
- 标记相对不确定度 > 50% 的参数
- 重点关注：Re、n、星等、b/a、PA

### 3. 边界触碰检查
- 检查 n 是否接近典型边界（0.1 或 20）
- 检查 Re 是否接近 0 或异常大
- 检查 b/a 是否恰好为 0 或 1
- 检查是否有参数值恰好等于常见约束值

### 第三步：综合诊断

如果存在问题，需要结合残差图像，逐项分析输入参数，找出不合理的初始值或参数范围设置，并提出具体的修改建议（如调整初始值、修改参数范围、合并或删除成分）。
如果不存在以上三项问题，请直接给出整体拟合质量的结论（PASS），并简要说明理由。
