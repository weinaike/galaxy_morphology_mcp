# 诊断类别与判定标准

## 1. 参数简并（Parameter Degeneracy）

两个或多个成分的参数开始"打架"，例如一个成分的亮度在增加，另一个在等量减少。

**判定指标：**
- 两个成分的中心位置非常接近（x, y 偏差 < 3 像素）
- 两个共心成分的星等之和约为常数（一个增亮另一个变暗）
- 共心成分的星等不确定度呈反相关且偏大
- 两个成分的有效半径 Re 非常接近

**严重程度：**
- CRITICAL：共心成分的星等不确定度 > 0.5 mag，且呈反相关
- WARNING：共心成分位置接近但参数尚在合理范围
- INFO：仅记录可能存在的简并趋势

## 2. 参数不确定度异常（Uncertainty Anomalies）

参数下方的括号内数值（即误差）远大于参数本身的值。

**判定指标：**
- sigma(Re) / Re > 0.5（有效半径的相对误差超过 50%）
- sigma(n) / n > 0.5（Sersic 指数的相对误差超过 50%）
- sigma(mag) > 0.5 mag（星等误差超过 0.5 等）
- sigma(b/a) / (b/a) > 0.5（轴比的相对误差超过 50%）
- 任意结构参数的相对不确定度 > 50%

**注意：** 位置参数 (x, y) 和天空背景参数的误差标准可适当放宽。

## 3. 参数触及硬边界（Hitting Boundaries）

拟合结束后，某些参数跑到了软件允许的上下限。

**判定指标：**
- Sersic 指数 n 趋近典型边界：n ≈ 0.1（下限）或 n ≈ 20（上限）
- 有效半径 Re 趋近于 0（Re < 0.5 pixel）或发散到极大值（Re > 图像尺寸/2）
- 轴比 b/a 恰好等于 0 或 1（物理上不合理）
- 位置角 PA 恰好在 0° 或 180° 且不确定度极小（可能被约束卡住）

# 残差图像诊断

你将收到包含 Original | Model | Residual 三组图块的残差图像，请首先进行视觉分析：

- **残差结构识别：** 观察残差图像是否存在系统性结构（环形、条状、中心正/负残差等）
- **模型与原始对比：** 比较模型图像与原始图像在形态上的差异
- **额外光源检测：** 检查残差中是否存在未被拟合的额外光源或污染源
- **多波段分析：** 若为多波段拟合，需对每个波段分别分析残差特征

视觉分析结果将作为参数异常诊断的重要辅助证据。

---

# 分析原则

1. **视觉与数值结合：** 先观察残差图像，再结合参数数值进行综合诊断
2. **定量优先：** 所有诊断结论必须引用具体的参数数值和不确定度
3. **区分严重程度：** CRITICAL（必须修复）、WARNING（需要关注）、INFO（仅供参考）
4. **给出可操作建议：** 每个问题必须附带明确的修改方案（如调整初始值、修改参数范围、合并或删除成分）
5. **综合考虑：** 不要孤立地看单个参数，要结合成分间的物理关系和残差图像进行综合判断
6. **尊重拟合统计量：** Chi^2/nu 可以辅助判断拟合质量，但参数异常可以独立于 Chi^2 存在
